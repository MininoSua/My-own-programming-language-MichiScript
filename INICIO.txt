🚀 CÓMO EMPEZAR:

1. REPL INTERACTIVO:
   $ python3 michiscript.py
   
   michis> traer "¡Hola Michis!"
   ¡Hola Michis!
   
   michis> x = 5 + 3
   michis> traer x
   8

2. EJECUTAR ARCHIVO:
   $ python3 michiscript.py archivo.michis

3. EJECUTAR EJEMPLOS:
   $ python3 michiscript.py ejemplos_compilador.michis

4. EJECUTAR PRUEBAS:
   $ python3 -m unittest test_michiscript

🎯 FLUJO DE USO:

   Código MichiScript
        ↓
    [LEXER]        (tokenización)
        ↓
    Token Stream
        ↓
    [PARSER]       (análisis sintáctico)
        ↓
    AST (Árbol)
        ↓
    [EVALUATOR]    (ejecución)
        ↓
    Resultado

💡 EJEMPLO SIMPLE:

   michiscript:
   ─────────────────────────────
   funcion saludar(nombre) {
       traer "Hola, " + nombre + "!"
   }
   
   saludar("Michi")
   
   Resultado:
   ─────────────────────────────
   Hola, Michi!

🐱 CARACTERÍSTICAS ESPECIALES:

   • Palabras clave temáticas:
     - "miau" para comentarios
     - "traer" para print
     - "pedir" para input
     - "rascar" para break
     - "ronronear" para continue
     - "gatito" para clases
     - "olfatear" para importar

   • Operadores intuitivos:
     - Aritméticos: + - * / % ^
     - Comparación: == != < > <= >=
     - Lógicos: && || !

   • Estructuras claras:
     - si/sino/elif para condicionales
     - mientras para bucles
     - para...en para iteración

🔧 REQUISITOS:
